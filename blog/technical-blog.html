Borders, Margins and Padding

Margin, padding and borders all make up parts of what is called the box model in CSS. This CSS model is then paired with an html element which would make up what is called the content inside the box at the centre. They all relate to each other and define space inside and outside the border. Pictured below is this concept visually.

The margin is what is responsible for defining the space between the outside of the border and other would be objects around it. Because of this it can also be used to position boxes around the page. The margin length can be defined on any of its four sides using pixels and other measurements.

On the inside of the margin lies the border which defines the perimeter of the box. The borders make up such as width and type can also be defined using measurements, and singular sides can also be individually manipulated.

At the centre box is content, on the inside between content and the box walls is the padding. Padding can control the position of the content within the box and can be used to manipulate aspects such as the visibility of the background colour. Measurements such as pixels are also used to apply a chosen amount of padding on the top, bottom or sides.

With all these components available to be adjusted the box model become very useful to know when laying out webpages and achieving desired visual effects.
